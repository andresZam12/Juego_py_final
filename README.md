# Juego_py_final

## Descripción
FPS básico en Unity donde el jugador debe sobrevivir y eliminar enemigos para avanzar de nivel. Incluye sistema de disparo, barra de vida, enemigos con IA, score y reinicio automático al morir.

## Características principales
- Movimiento y control tipo FPS
- Disparo con raycast y sonido
- Enemigos con IA (persiguen y atacan al jugador)
- Barra de vida visual
- Sistema de puntuación y avance de nivel
- Reinicio automático tras muerte

## Controles
- **WASD**: Moverse
- **Mouse**: Mirar
- **Click Izquierdo**: Disparar
- **Shift**: Sprint
- **Space**: Saltar

## Estructura de carpetas
- `Assets/Scenes/MundoUno/Scripts/` — Scripts principales del juego
- `Assets/Scenes/EcenesEJEC/` — Escenas del juego
- `Assets/Scenes/MundoUno/` — Prefabs y objetos de la escena principal
- `Assets/Scenes/MundoUno/Scripts/UI/` — Scripts de interfaz
